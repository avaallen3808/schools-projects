# CMS Sekolah - PRD Documentation v2.1.0

Dokumentasi Product Requirements Document (PRD) untuk CMS Sekolah.

## Tech Stack

- **Frontend:** Next.js 15 + React 19 + TypeScript
- **Backend:** NestJS 10 + Prisma 6
- **Database:** PostgreSQL 16 + Redis 7
- **Monorepo:** Turborepo 2 + pnpm 9

## Quick Start

1. Extract this folder
2. Read PRD.md for full requirements
3. Copy templates/ folder as your monorepo starter
4. Run: cd templates && pnpm install
5. Setup DB: docker-compose up -d
